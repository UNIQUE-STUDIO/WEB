## Features

- **28 Premium Templates** - Modern designs with real preview URLs
- **PWA Support** - Installable, offline-capable with service worker
- **SEO Optimized** - Open Graph, Twitter Cards, schema.org JSON-LD
- **Multi-language** - English and Russian interfaces
- **Admin Panel** - Leads, customers, orders, analytics, backup
- **VK Integration** - Automated lead notifications via VK API
- **Cloud Sync** - GitHub cloud storage for all data
- **Dark Mode** - Built-in dark/light theme toggle
- **Responsive** - Mobile-first design, all screen sizes

```bash
cd WEB

```
